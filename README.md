## About this repository

This repository documents my hands-on studies from the **3rd edition** of *"Automate the Boring Stuff with Python"* by Al Sweigart.

Focuses on **real-world task automation** examples from the book. Code is organized by **topic** — making it easier to quickly find what you need.
